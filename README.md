Automatically converts URLs from CSV files into Obsidian notes with Markdown formatting.

## Features
- Batch processing of hundreds of URLs
- Metadata preservation (tags, titles, descriptions)
- Fallback to Archive.org for blocked content
- Obsidian-ready format with YAML frontmatter
- Detailed success/failure reports
